# Day 06 Learnings

## What I Learned

* Aliases (`AS`) provide temporary names to columns or expressions.
* Aliases improve the readability of query results.
* Using `AS` does not modify the original table.
* SQL can perform calculations directly inside the `SELECT` statement.
* Calculated columns can also have aliases.

## Confidence Check

✅ I can rename columns using `AS`.

✅ I can create calculated columns such as `salary * 12 AS Annual_Salary`.

✅ I understand that aliases only affect the query output.

## Interview Takeaway

Aliases make SQL queries easier to understand and are commonly used in reports and dashboards. They help present meaningful column names without changing the database structure.

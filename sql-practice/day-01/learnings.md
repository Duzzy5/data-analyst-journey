# Day 01 Learnings

## What I Learned

- A database is a collection of related tables.
- A table stores information in rows and columns.
- The SELECT statement retrieves data from a table.
- The order of columns in the output depends on the SELECT statement, not the table structure.

## What Confused Me

- Why SELECT * is discouraged in large databases.
- Which data type is best for storing salary (INT vs FLOAT vs DECIMAL).

## Mistake I Made

- I initially thought a table was just a format instead of an object that stores data.

## One Thing I Won't Forget

Always think:

**Business Question → SQL Query**

instead of memorizing syntax.

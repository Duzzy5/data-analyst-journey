# Day 05 Learnings

## What I Learned

* `DISTINCT` returns only unique values from a column.
* It removes duplicate entries from the result.
* `DISTINCT` can be used with one or multiple columns.
* When multiple columns are selected, SQL checks the uniqueness of the entire combination.
* `DISTINCT` can be combined with `ORDER BY` to display unique values in a sorted order.

## Confidence Check

✅ I can retrieve unique values using `DISTINCT`.

✅ I understand the difference between `SELECT` and `SELECT DISTINCT`.

✅ I can combine `DISTINCT` with `ORDER BY`.

## Interview Takeaway

`DISTINCT` is commonly used during Exploratory Data Analysis (EDA) to understand the unique values present in a dataset before performing further analysis.

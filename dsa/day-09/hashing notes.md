## Key Learnings

- HashMap stores data as Key → Value.
- Searching a key is approximately O(1).
- Frequency counting is one of the most common HashMap patterns.
- Use:
  - `key in dictionary`
  - `dictionary[key]`
  - `dictionary.items()`
- Don't use `dict` as a variable name because it is a built-in Python type.
